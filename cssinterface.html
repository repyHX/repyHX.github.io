Interface properties CSS

Term: Box Shadow

Purpose:Adds a shadow effect around an elements frame

Example:

box-shadow: 60px -16px teal;

Term:Border Radius

Purpose: It makes the elements frames' corners rounded

Example:

border-radius: 30px;

Term: Colors

Purpose:Helps select a color for the text

Example:

color:Tomato;

Term: Fonts

Purpose:A selector for the font

Example:

font-family: Arial, Helvetica, sans-serif;

Term: Lists

Purpose:Style a list with this property

Example:

list-style-type: circle;

Term: Background

Purpose:To style the background

Example:

background: green;
