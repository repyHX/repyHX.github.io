Media Query CSS

Media Query

Term: Media Screen

Purpose:This targets for devices using screens

Example:

@media screen and (max-width: 600px) div.example { display: none; }

Term: Media print

Purpose: To print preview the page

Example:

@media print { body { font-size: 10pt; } }

Feature Query

Term: Support Query

Purpose:A way for developers to test if a website supports certain features

Example:

@supports not (row-gap: 10px) { .box { border: 4px solid red; color: red; } }

Term: Support Query

Purpose:Tests if website supports css grid.

Example:

@supports (display: grid)

Import

Term: Import URL

Purpose:A source to pull resources from

Example:

@import url list-of-media-queries;

Term: Import Font

Purpose:

Example:

@font-face { font-family: myFirstFont; src: url(sansation_light.woff); }

Font face

Term: Source

Purpose:Give the CSS file a font file path to pull the data from

Example:

src: url(sansation_bold.woff);

Term: Font Face

Purpose:To tell the css page to select a non-native font

Example:

@font-face
