Box Model CSS

Term: Positioning

Purpose:Tells CSS on the method we use to position the elements

Example:

position: relative;

Term: Box Sizing

Purpose:sets how the elements width and height is presented

Example:

box-sizing: border-box;

Term: Margin

Purpose:To create space around the element

Example:

margin: 100px;

Term: Border

Purpose:To outline the elements box size

Example:

border: black;

Term: Padding

Purpose:To create space between the elements near the target element

Example:

padding: 5px;
